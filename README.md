# vocab  计算机术语词库
A vocabulary tool for computer science terms with categorization, search, and favorite features.

## 📖 词库来源
本项目的原始词库（术语和翻译）来源于 [EarsEyesMouth/computerese-cross-references](https://github.com/EarsEyesMouth/computerese-cross-references)，感谢原作者的整理和分享。

## ✨ 我的贡献
- 为每个术语添加详细解释
- 将数据整理为结构化的JSON格式
- 构建Web界面便于浏览和学习
